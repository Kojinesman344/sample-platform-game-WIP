## Features
- Classic platformer gameplay inspired by Megaman
- Multiple levels with increasing difficulty
- Unique character abilities
- Various enemies with distinct behaviors
- Engaging boss fights

## Controls
- **Left Arrow / A**: Move Left
- **Right Arrow / D**: Move Right
- **Up Arrow / W**: Jump
- **Space**: Shoot / Attack
- **ESC**: Pause Menu
